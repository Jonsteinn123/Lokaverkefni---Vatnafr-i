This folder contains the shape file and GeoPackage of the EU-Hydro network, clipped to Iceland, in EPSG:3057.

Description of the attributes:

--------------------------------

30 attributes, see documentation at:

EEA: EU-Hydro – River Network Database, Version 1.2, European Environment Agency under the framework of the Copernicus programme, available at: https://land.copernicus.eu/imagery-in-situ/eu-hydro/eu-hydro-river-network-database (last access: 06.02.2023), 2019.
